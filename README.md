Захар Рогачев GUI-приложение «Random Password Generator» с возможностью настройки параметров пароля, сохранением истории и Git.                                                                                  Использование
Установите желаемую длину пароля с помощью ползунка.
Отметьте галочками типы символов.
Нажмите кнопку «Сгенерировать пароль».
Сгенерированный пароль появится в поле ввода. Нажмите «Скопировать», чтобы поместить его в буфер обмена.
